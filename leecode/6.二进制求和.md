# [力扣67](https://leetcode.cn/problems/add-binary/description/)

给你两个二进制字符串 a 和 b ，以二进制字符串的形式返回它们的和。

例 1：

输入:a = "11", b = "1"

输出："100"




```python
class Solution:
    def addBinary(self, a: str, b: str) -> str:
        res = ''
        carry = 0
        p1,p2 = len(a)-1,len(b)-1
        while p1>=0 or p2>=0 or carry:
            total = 0
            if p1>=0:
                total += int(a[p1])
                p1 -= 1
            if p2>=0:
                total += int(b[p2])
                p2 -= 1
            total += carry

            cur = str(total %2)
            carry = total//2

            res = cur + res
        return res

```
