# [力扣445](https://leetcode.cn/problems/lMSNwu/description/)

给你两个 非空 链表来代表两个非负整数。数字最高位位于链表开始位置。它们的每个节点只存储一位数字。将这两数相加会返回一个新的链表。

你可以假设除了数字 0 之外，这两个数字都不会以零开头。

例1：

输入：l1 = [7,2,4,3], l2 = [5,6,4]
输出：[7,8,0,7]

进阶：如果输入链表不能翻转该如何解决？

```python
# 按照之前的思路,可以先将两个链表分别反转再相加,最后再反转即是答案
class Solution:
    def addTwoNumbers(self, l1: ListNode, l2: ListNode) -> ListNode:
        def reverse(head):
            pre,cur = None,head
            while cur:
                tmp = cur.next
                cur.next = pre
                pre = cur
                cur = tmp
            return pre
        l1 = reverse(l1)
        l2 = reverse(l2)

        def addTwoNumbers(l1,l2):
            dummy = p0 = ListNode(0)
            carry = 0
            while l1 or l2 or carry:
                total = 0
                if l1:
                    total += l1.val
                    l1 = l1.next
                if l2:
                    total += l2.val
                    l2 = l2.next
                total += carry

                p0.next = ListNode(total%10)
                p0 = p0.next
                carry = total//10
            return dummy.next
        return reverse(addTwoNumbers(l1,l2))

```

# 如果不能对链表实施反转操作
class Solution:
    def addTwoNumbers(self, l1: Optional[ListNode], l2: Optional[ListNode]) -> Optional[ListNode]:
        nums1,nums2 = [],[]
        while l1:
            nums1.append(l1.val)
            l1 = l1.next
        while l2:
            nums2.append(l2.val)
            l2 = l2.next

        carry = 0
        tail = None

        while nums1 or nums2 or carry:
            total = 0
            if nums1:
                total += nums1.pop()
            if nums2:
                total += nums2.pop()
            total += carry    
            carry = total // 10
            cur = ListNode(total%10)
            cur.next = tail
            tail = cur

        return tail
